Установка GLScene VCL
в Embarcadero RAD Studio

Инструкция по установке библиотеки GLScene VCL в Embarcadero RAD Studio для работы в Delphi & C++Builder. Выполните следующие шаги:

1. Загрузите исходные коды библиотеки GLSceneVCL из репозитория с URL https://svn.code.sf.net/p/glscene/code/trunk сайта http://sourceforge.net/projects/GLSceneVCL в директорию D:\GLScene\SVN_GLSceneVCL на вашем диске.

Для этого можно использовать программу TortoiseSVN (команда svn checkout) или встроенную в среду RAD Studio систему поддержки версий, вызываемую из меню File | Open From Version Control…

Также можно загрузить текущую версию GLScene в виде снимка архива со страницы http://sourceforge.net/p/glscene/code/HEAD/tree/ . Для этого в правой части вверху страницы необходимо кликнуть по кнопке Snapshot.

Официальный последний релиз библиотеки можно  скачать из раздела  http://sourceforge.net/projects/glscene/files/

 

2. Скопируйте папку trunk в отдельную рабочую директорию с именем D:\GLScene\GLSceneVCL, чтобы исключить случайное изменение исходных файлов.

 

3. Запустите командный файл setup.bat, находящийся в директории  D:\GLScene\GLSceneVCL\external, с правами администратора и скопируйте в Windows\System32 и Windows\SysWOW64 вспомогательные внешние динамические библиотеки 3D звука, шейдеров и физики (BASS, FMOD, OpenAL, SDL, CG, ODE, Newton). Иначе вам придется для каждого приложения, обращающегося к этим библиотекам, помещать соответствующие DLL в директории с исполняемыми файлами. Замечание: если при запуске setup64.bat на OS Windows 10 возникнут проблемы с доступом к системным папкам, то вручную скопируйте соответствующие файлы *.dll.

 

4. Опции Delphi для доступа к исходным модулям. Если вы впервые устанавливаете библиотеку GLSceneVCL, то необходимо открыть раздел Environment Variables в диалоге Options добавить путь к пакетам Bpl создать новую переменную среды GLSVCLDIR, как показано на следующем скриншоте:

 

Теперь откройте диалог Delphi Options Library

 

и добавьте в пункте Library Path пути к исходным файлам GLScene, расположенным в директории, присвоенной переменной $(GLSVCLDIR)\:

 

 

 

5. Опции C++ Builder для файлов заголовков HPP. Откройте диалог Options в меню Tools. Добавьте в строке System Include path в секции Paths and Directories путь $(BDSCOMMONDIR)\hpp\$(Platform) к заголовочным hpp файлам, генерируемым при компиляции исходников GLScene:

 

 

 

6. Опции C++ Builder для файлов библиотек BPI/LIB. Откройте диалог Options в меню Tools. Добавьте путь для доступа C++ приложений GLScene к библиотечным файлам  $(BDSCOMMONDIR)\Dcp  в строке “Ordered list of Library paths”:

 

 

7. Откройте через меню File|Open Project…(Ctrl+F11)  групповой проект  GLSceneVCL.groupproj, находящийся в директории $(GLSVCLDIR)\Packages. В окне Project Manager должны быть перечислены следующие проекты с расширением bpl:

 

 

8. Проверьте правильность установки опции “Generate all C++Builder files (including package libs)” -  сгенерировать все C++Builder файлы (включая библиотеки libs/bpi). По умолчанию эта установка указана в файлах проектов *dproj для всех пакетов GLScene:

 

 

9. Для подключения модулей DesignEditors, DesignIntf, ToolsAPI и VCLEditors в последних компиляторах, как указано в справке RAD Studio,  необходимо добавить опцию Delphi  компилятора  -LUDesignIDE  на закладке Compiling в диалоге Project Options:

 

 

10. Cкомпилируйте все пакеты группового проекта GLSceneVCL. Для этого в окне Project Manager щелкните правой кнопкой мыши по верхней строке проекта GLSceneVCL и выполните команду  Compile All From Here.  После компиляции установите DesignTime пакеты с визуальными компонентами в палитру компонентов RAD Studio с помощью команды Install.

(Замечание: некоторые пакеты не включены в групповой проект. Пакет GLSceneVCL_SDL.dpk может быть откомпилирован и использоваться в режиме RunTime, однако конфликтует в новой средой разработки в режиме DesignTime, поэтому временно не устанавливается в палитру компонентов, пакет GLSceneVCL_DWS требуют подключения вспомогательных библиотек, отсутствующих в данной версии GLSceneVCL).

 

После установки компонентов GLSceneVCL можно приступить к работе с библиотекой и выполнить демонстрационные примеры для Delphi и C++Builder, находящиеся в директории $(GLSVCLDIR)\Samples.
